## About Me

- I'm an **AI/ML Engineer** based in **Seattle, WA**, focused on building intelligent systems that push the boundaries of what LLMs can do.
- I specialize in **Large Language Models (LLMs)**, **AI Agents**, and **Agentic Systems** — from research benchmarking to production-grade deployments.
- I build things end-to-end: RAG pipelines, multi-agent frameworks, knowledge graphs, medical AI, and cloud-native LLM gateways.
- Early adopter of **Model Context Protocol (MCP)** and **Google ADK** for next-generation agent architectures.
- I benchmark rigorously — my LLM Hallucination Benchmark evaluated **10 open-source models across 400 Q&A pairs** with dual-metric scoring.
- I believe in **open-source AI** and share reusable, production-ready tools with the community.

---

## What I'm Currently Working On

- **Agentic Systems** — designing multi-agent workflows using Google ADK and MCP for real-world automation tasks
- **LLM Evaluation Frameworks** — extending hallucination benchmarks to cover reasoning, factuality, and tool-use accuracy
- **Production LLM Infrastructure** — building scalable, secure gateways and RAG pipelines with proper observability
- **Biomedical AI** — expanding knowledge graph capabilities and integrating LLM reasoning over structured medical data
